Distributed Video
Transcoding Platform

StreamForge is a full-stack video processing platform that accepts raw uploads and automatically transcodes them into adaptive HLS streams — the same format used by YouTube, Netflix, and Twitch. Built with Django, Celery, Redis, and FFmpeg.

Upload Any Video

Drag and drop any video format. Django REST Framework handles ingestion and queues the job.

Adaptive HLS Output

Celery workers run FFmpeg to segment raw uploads into 6-second HLS renditions (1080p and 720p) concurrently.

Sprite Hover Animation

Hovering video tiles runs lightweight CSS background-position loop previewing automatically generated frame-sprites.

Adaptive Quality Transcoding specs

FFmpeg output configurations parsed by the worker nodes during ingestion pipeline.

HLS 1080p

6s segments · 5.0 Mbps

High resolution layout optimized for desktop viewports running on broad bandwidth connections.

HLS 720p

6s segments · 2.8 Mbps

Balanced stream configuration providing highly visual quality with reduced cellular data requirements.
