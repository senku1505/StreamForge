StreamForge is a full-stack video processing platform that accepts raw uploads and automatically transcodes them into adaptive HLS streams — the same format used by YouTube, Netflix, and Twitch. Built with Django, Celery, Redis, and FFmpeg.

Upload Any Video

Drag and drop any video format. Django REST Framework handles chunked ingestion and queues the job immediately.

Auto-Transcoded via Celery

Celery workers pick up the job from Redis and run FFmpeg to produce HLS segments at 1080p and 720p, extract a thumbnail, and build a sprite sheet.

Stream with Video.js

Video.js loads the HLS master playlist and adaptively switches quality. The seek bar shows a sprite-based thumbnail preview on hover.

Processing Pipeline

End-to-end flow from upload to stream

1 Upload

Client POSTs file to /api/videos/ via XHR with progress tracking

2 Queue

Django saves the raw file and pushes a process_video task to Redis

3 Transcode

Celery worker runs FFmpeg to produce HLS 1080p + 720p, thumbnail, and sprite sheet

4 Stream

Video.js loads master.m3u8 and adaptively switches between quality renditions
